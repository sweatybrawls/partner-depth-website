How to choose

Most engagements start with the audit.

If you already know what's broken, skip the audit and go straight to Strategy — we'll design the fix. If you have a clear plan but not the headcount, start with Growth — we'll execute against it.

Most programs are somewhere in between. The audit gives both sides a shared, data-backed view of the program before either of us commits to scope. From there, the next move is usually obvious.

  • Don't know what's wrong? Start with Audit.
  • Know the problem, need a plan? Start with Strategy.
  • Have a plan, need execution? Start with Growth.
  • New program, blank page? Strategy then Growth.

Not sure which one fits?

Thirty minutes. We'll talk through your program and tell you what we'd actually recommend — even if it isn't us.
